# Morning Short Paper Bot v1.0

09:15 국내 교차상장 급등 1등 종목 숏 모의투자 봇.

## 실행
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```

## 핵심 전략
- 09:15 KST 실행
- 업비트 + 빗썸 교차상장
- 비트겟 선물 가능
- +3% 이상 급등 종목 중 상승률 1등 1개만 선택
- PAPER 숏 진입
- 1차 2%, 2차 1%, 3차 1%
- 2차: 1차 대비 +5%
- 3차: 2차 대비 +5%
- TP: 레버리지 기준 +12% 전량청산
- SL: 16:00 이후 -30% 이하일 때 전량청산


## v1.1 수정
- 기존 24시간 상승률 기준 제거
- 09:00 기준가 저장
- 09:15 현재가와 비교
- 09:00 → 09:15 사이 15분 급등률 기준으로 +3% 이상 필터
- 그중 가장 많이 튄 종목 1개만 PAPER 숏 진입


## v1.2 수정
- 텔레그램 메뉴에 수동 테스트 버튼 추가
  - 🧪 기준가 저장 테스트
  - 🧪 15분 스캔 테스트
- 내일 아침까지 기다리지 않고 스캔 로직을 바로 검증 가능


## v1.3 수정
- 거래소 디버그 모드 추가

  - 업비트 KRW 종목 수
  - 빗썸 KRW 종목 수
  - 비트겟 선물 계약 수
  - 비트겟 티커 수
  - 업비트∩빗썸 교집합 수
  - 업비트∩빗썸∩비트겟선물 최종 교집합 수
- Bitget 심볼 정규화 추가

  - 1000PEPEUSDT → PEPE
