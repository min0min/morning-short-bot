# Morning Short Paper Bot v1.5 Final

## 핵심 전략

- 09:00 KST 기준가 저장
- 09:00~09:15 동안 30초마다 가격 갱신
- 업비트 KRW + 빗썸 KRW + 비트겟 USDT 선물 교차상장 종목만 추적
- 15분 동안의 최고 상승률 기준으로 +3% 이상 급등 후보 선정
- 후보 중 최고 상승률 1등 종목 1개만 PAPER 숏 진입
- 1차: 시드 2%
- 2차: 1차 진입가 대비 +5% 상승 시 시드 1%
- 3차: 2차 진입가 대비 +5% 상승 시 시드 1%
- 레버리지: 4배 기준 계산
- TP: 레버리지 기준 +12% 전량 청산
- SL: 16:00 이후 레버리지 기준 -30% 이하이면 전량 손절, 아니면 홀딩

## 테스트 버튼

- 🔍 거래소 디버그
- 🧪 기준가 저장 테스트
- 🧪 최고가 갱신 테스트
- 🧪 피크 스캔 테스트
- 🧪 오늘 급등 테스트

## 실행

```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```

## Railway 환경변수

```env
TELEGRAM_BOT_TOKEN=새토큰
TELEGRAM_CHAT_ID=채팅ID
PAPER_SEED_USDT=1000
MODE=PAPER
```

현재 버전은 실주문 없는 PAPER MODE 전용입니다.
